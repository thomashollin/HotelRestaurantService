1. Xampp (Windows), Mampp (MacOS) installieren
2. Repository in Xampp/htdocs kopieren
3. Xampp starten
4. In Xampp (Browser) -> Tools -> phpmyAdmin Datenbank "db_restaurant_service" erstellen
5. User "hotelguest" mit PW "hotelguest" anlegen und Berechtigung vergeben
6. In phpmyAdmin unter Datenbank -> SQL sql_skript_ohne_Tischnummer kopieren und ausführen
7. Service kann unter http://localhost:8888/hotelrestaurantservice/src/index.php gestartet werden

--Änderungen können direkt im htdocs Ordner vorgenommen werden--
